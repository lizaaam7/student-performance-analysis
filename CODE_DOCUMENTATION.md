### 6.1 T-Test (—Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –¥–≤—É—Ö –≥—Ä—É–ø–ø)
```python

group1 = df[df['test_preparation_course'] == 'completed']['average_score']
```

**–ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è:**
- `t_stat` ‚Äî t-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ (—á–µ–º –±–æ–ª—å—à–µ –ø–æ –º–æ–¥—É–ª—é ‚Äî —Ç–µ–º —Å–∏–ª—å–Ω–µ–µ —Ä–∞–∑–ª–∏—á–∏–µ)
- `p_value` ‚Äî –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª—É—á–∏—Ç—å —Ç–∞–∫–∏–µ –¥–∞–Ω–Ω—ã–µ —Å–ª—É—á–∞–π–Ω–æ

**–ü—Ä–∞–≤–∏–ª–æ:**
- p < 0.05 ‚Üí **–°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏ –∑–Ω–∞—á–∏–º–æ** (—Ä–∞–∑–ª–∏—á–∏–µ —Ä–µ–∞–ª—å–Ω–æ–µ)
- p ‚â• 0.05 ‚Üí –ù–µ –∑–Ω–∞—á–∏–º–æ (—Ä–∞–∑–ª–∏—á–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–ª—É—á–∞–π–Ω—ã–º)
